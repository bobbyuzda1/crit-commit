# Crit Commit: Game Design Guide

A player-facing guide to all game mechanics in Crit Commit.

---

## Classes

Choose a class during character creation. Each class earns bonus XP from different coding activities.

| Class | Archetype | Strength | Passive Ability |
|-------|-----------|----------|-----------------|
| **Architect** | Backend/systems thinker | Bonus XP from Bash commands and complex file edits | **Blueprint** — quest objectives are revealed more clearly |
| **Scout** | Debugger/investigator | Bonus XP from Read, Grep, and search patterns | **Tracker** — higher rare card drop rate, natural crit chance bonus |
| **Artificer** | Builder/creator | Bonus XP from Write and new file creation | **Craftsmanship** — better gear rarity on drops |
| **Battlemage** | Full-stack generalist | Flat XP from all activities (no bonus, no penalty) | **Versatility** — can equip any materia type |

---

## Materia

Materia are equippable coding skill orbs that level up through use. Equip them to gain passive bonuses.

| Type | Color | What It Tracks | Examples |
|------|-------|----------------|----------|
| **Skill** | Green | Languages and frameworks detected in sessions | TypeScript, Python, React, SQL |
| **Tool** | Yellow | Developer tool usage from your coding activity | Terminal, Search, Editor, Git |
| **Spirit** | Red | AI-specific activity — sub-agents, complex prompts | Summoner, Strategist, Architect |

Materia level from 1 to 5 by accumulating AP (Activity Points) from relevant coding work. Higher levels unlock stronger passive bonuses.

## Crit System

Crits are the signature mechanic. Exceptional coding outcomes trigger critical hits with bonus rewards.

### Crit Types

| Coding Achievement | Crit Type | Effect |
|---|---|---|
| All tests pass on first run | **Crit Strike** | 3x XP, doubled rare item drop chance |
| Fix a bug (tests red to green) | **Crit Recovery** | Bonus XP + party morale boost (temporary XP multiplier) |
| Push/deploy with zero errors | **Crit Deploy** | Boss-finishing animation, guaranteed loot drop |
| 60+ minute focused session | **Crit Focus** | Limit Break gauge fills instantly |
| First commit of a new project | **Crit Genesis** | New zone discovery bonus, unique materia seed |
| Merge a large PR / resolve conflicts | **Crit Merge** | Party fusion attack — all companions strike together |
| Sub-agent completes successfully | **Crit Summon** | Summoned creature delivers double rewards |

### Crit Streak

Land **3 Crits in a single session** to trigger a Crit Streak — a temporary buff where all XP is doubled and every encounter drops loot for the rest of that batch interval.

### Crit Chance

Starts at ~5% at level 1, grows to ~25% at level 20. The Scout class has a natural bonus. Gear and materia can increase it further.

## Quests

Three tiers of quests run simultaneously, from quick background tasks to multi-session story arcs.

### Tier 1: Micro-Quests (3 slots, always active)

Generated automatically from your coding patterns. Zero AI cost.

| Pattern | Quest | XP |
|---|---|---|
| 5 file edits | "Edit 5 files" | 10-25 |
| 3 search operations | "Investigate the codebase" | 10-25 |
| Run tests | "Test your mettle" | Scales with results |
| First commit of the day | "Dawn Patrol" | 25 |
| Use a new language/framework | "Foreign Lands" | 25 |

Micro-quests auto-rotate when completed. They're always ticking in the background.

### Tier 2: Session Quests (2 slots)

Generated by Claude each batch interval. Narrative-driven and contextual to your current work.

> *"The Legacy Module Caverns grow darker. Three test failures echo through the halls — the Bug Swarm grows stronger. Defeat them by making all tests pass before your session ends."*

**XP:** 50-150 + item drops.

### Tier 3: Epic Quests (1 slot)

Triggered by large-scale work patterns — new feature branches, major refactors, migrations. Multi-stage, spanning multiple sessions.

> *"The Great Migration — Chapter 1 of 4: The old database schema crumbles. Begin the journey by creating the new models."*

**XP:** 200-500 per stage. Rare rewards on completion.

### Stale Quest Retirement

Quests that become irrelevant retire with narrative flavor:

> *"The Bug Swarm retreated into the shadows... for now."*

## Zones

Your world map is unique — zones generate from your personal coding patterns.

### Cloud City Base Camp (permanent)

Your home hub. Always available. Contains:
- Character and party NPCs
- Quest board
- Stackjack card table
- Inventory and gear management
- Coffee shop

### Dynamic Zones

When enough coding activity accumulates in a category, the game engine generates **two zone options** with different gameplay modifiers. You choose one.

| Coding Activity | Possible Zone | Flavor |
|---|---|---|
| Heavy Python work | The Serpent Gardens | Winding vine-covered paths, snake creatures, green mist |
| Database queries | The Crystal Vaults | Underground caverns, data crystals, table-shaped formations |
| React/frontend work | The Prism Towers | Colorful spires, component architecture, light refraction |
| DevOps/CI-CD | The Pipeline Forge | Industrial, flowing lava channels, automated machinery |
| Heavy Git operations | The Branching Wilds | Forest where paths fork and merge, timeline distortions |
| Security/auth work | The Shadow Keep | Dark fortress, locked gates, key-based puzzles |
| API/integration work | The Bridge Nexus | Floating bridges connecting islands, message carriers |

### Zone Limits

Maximum **6 active zones** + Cloud City Base Camp. If a 7th zone unlocks, choose one to archive — it stays on the map (grayed out) and can be revisited but no longer generates encounters.

## Progression

### Leveling

| Level Range | Approximate Time | Design Intent |
|---|---|---|
| 1-5 | First day | Hook you fast |
| 6-10 | ~1 week | Zones unlocking, materia leveling |
| 11-15 | ~2-3 weeks | Epic quests drive progression |
| 16-20 | ~1-2 months | Levels feel like real achievements |

### XP Sources

| Activity | XP | Notes |
|---|---|---|
| Micro-quest completed | 10-25 | Background steady drip |
| Session quest completed | 50-150 | Main progression driver |
| Epic quest stage | 200-500 | Big milestone reward |
| Tests passing | 5-30 | Scales with count |
| Bug fixed (red to green) | 30-75 | Rewards persistence |
| Git push | 20-40 | Shipping matters |
| Stackjack win | 15-35 | Side activity reward |
| New zone unlocked | 100 | Exploration bonus |
| First session of the day | 25 | Daily login bonus |
| Return after 2+ days | 50 | Rested XP (anti-burnout) |

### Anti-Grind Protections

- Diminishing returns on repetitive actions
- The game engine detects suspicious patterns and addresses them narratively
- Rewards outcomes (tests passing, bugs fixed) over inputs (lines of code)

### Ascension (Prestige)

At **level 20**, you can Ascend:

- Reset to level 1
- Earn an **Ascension Star** (visible on your profile and character sprite)
- Choose a permanent passive bonus based on your playstyle
- Zones stay unlocked but encounters scale harder
- Ascension-only Stackjack cards become available
- Character sprite gets a visual upgrade (glow, enhanced armor)

You can Ascend multiple times. A 5-star Ascended player is a legend on the community leaderboard.

### Gear Rarity

| Tier | Drop Rate | Source |
|------|-----------|--------|
| Common | ~60% | Micro-quests, basic encounters |
| Uncommon | ~25% | Session quests, Stackjack wins |
| Rare | ~12% | Epic quests, zone bosses |
| Legendary | ~3% | Ascension rewards, major milestones |

## Party System

### Terminal Companions

Each active Claude Code terminal becomes a companion NPC. Claude generates each with a name blending pop culture characters with coding context.

| Terminal Activity | NPC Name | Inspiration |
|---|---|---|
| Running Python tests | Darth Pytest | Star Wars + pytest |
| TypeScript refactoring | Gandalf the Typed | LOTR + TypeScript |
| Database queries | Link of the Lost Tables | Zelda + SQL |
| Docker/containers | Morpheus Container | Matrix + Docker |
| React frontend work | Tifa Reacthart | FF7 + React |
| Git operations | Luigi Rebase | Mario + git rebase |
| Debugging | Frodo Debuggins | LOTR + debugging |

### Sub-Agent Summons

When Claude Code spawns a sub-agent, a summoned creature appears in the game world. It's short-lived and powerful — when the sub-agent finishes, the creature delivers its reward and disappears.

### Idle and Offline

- Terminals idle for 10+ minutes become "resting" NPCs at camp
- When no terminals are active, you can still manage inventory, play Stackjack, and browse the map
- When a terminal starts, the game resumes immediately

### Party Size

More terminals = bigger party = harder encounters available with better rewards. All activity feeds your character's progression.
